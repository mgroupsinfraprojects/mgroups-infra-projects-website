# `app/ui/` — Frontend Layer

PySide6 desktop interface. This layer must depend on `app.core`, but `app.core` must not depend on PySide6.

## Main responsibility

- Main window and page navigation
- Upload screen
- Dashboard and KPI cards
- Audit/review screen
- Supplier and reconciliation views
- Export/settings views
- Guided search and popup filters
- Themes, QSS, display settings
- Background worker integration
