# theme/

Readable facade for theme and display settings.

## Owns
- Professional Blue / Light / Dark / High Contrast theme choices.
- Font size and density defaults.
- Sidebar, card, button, and text color controls.

## Runtime source
- `app/ui/theme_manager.py`
- `app/assets/styles/main.qss`
- `app/assets/styles/*.qss`

## Rule
Theme changes must not hide mandatory audit warnings or export-safety status.
