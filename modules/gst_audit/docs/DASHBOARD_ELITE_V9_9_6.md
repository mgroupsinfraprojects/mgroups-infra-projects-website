# v9.9.6 Dashboard Elite Decision Release

This release improves only the dashboard-facing user experience while keeping the application small.

## Dashboard upgrades

- Decision-first top card: status, plain next step, and direct action buttons.
- Official vs visible totals: prevents users from confusing filtered values with official report totals.
- Fix-first issue queue: review rows, high risk, GST mismatch, duplicate/skipped rows.
- Quality Gate preview: row coverage, amount reconciliation, review status, export readiness.
- Charts remain secondary and collapsible.

## User flow

1. Choose files.
2. Start audit.
3. Read the dashboard status.
4. Click the issue that needs fixing.
5. Export only after the Quality Gate is safe.

## Scope

This is a source-package GUI design upgrade. Final Windows visual validation and EXE double-click validation still require a Windows machine.
