# GST Audit Pro v9.9.6 — Simple GUI Elite Release

This release consolidates the strongest parts found across the v9.9, v9.9.1, v9.9.2 Professional, v9.9.2 Configuration, and v9.9.3 releases.

## Merged best components

- v9.9.3: true one-file launcher and branding/navigation configuration.
- v9.9.2 Professional: standalone `app/core/quality_gate.py`, deterministic quality-gate scoring, and richer sign-off controls.
- v9.9.2 Configuration: configurable product identity and navigation labels.
- v9.9.1: stable release verification and public import safety baseline.
- v9.9: effectiveness/UI workflow improvements.

## Final launcher

Open only:

```bat
START_GST_AUDIT_PRO.bat
```

`run_app.bat` is intentionally removed.

## Remaining external validation
