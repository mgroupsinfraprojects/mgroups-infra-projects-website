# v9 Advanced CMS Report

## Added
- Per-field text styling controls beside public visibility controls.
- Font family, size, weight, color, alignment, italic, uppercase, letter spacing, line height.
- Visual page builder for homepage custom blocks.
- Draft/publish control for page builder blocks.
- Multi-admin roles: owner, editor, viewer.
- Viewer POST protection.
- Side-by-side version comparison using HTML diff.
- Drag ordering support extended to page builder blocks.
- Browser E2E smoke test plan and Playwright skeleton.

## Privacy defaults
- Phone and WhatsApp remain hidden by default.
- Project values remain private unless explicitly shown.
- Sensitive certificates, invoices, tax, PAN/Aadhaar/TAN and legal data are not displayed publicly.
