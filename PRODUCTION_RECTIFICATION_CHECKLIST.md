# Production Rectification Checklist

Use this checklist before sharing the website as a real company website.

## Must replace placeholder data

- Real phone number
- Real WhatsApp number
- Real email
- Real office/service address
- Real company profile
- Verified years of experience
- Verified completed/active project counts
- Real services only
- Real project photos
- Real project case details

## Must configure on Render

- `SECRET_KEY`
- `COOKIE_SECURE=1`
- `DATABASE_URL` from PostgreSQL
- `CLOUDINARY_URL` from Cloudinary

## Admin safety

- Change default password immediately
- Download backup before every major update
- Export enquiries regularly
- Keep project images compressed and named clearly

## Do not claim unless true

- ISO certification
- Pan-India operations
- Government contracts
- Exact project counts
- Exact team size
- Free site visits
